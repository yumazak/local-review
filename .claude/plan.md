# 背景
claude code を活用するにあたって、VS Code 上の差分ビュー上で GitHub のレビューと同様にコメントをできるようにしたい。

# 要件
- VS Code の差分ビュー上で、特定の行にコメントを追加できること
  - + ボタンを押すとコメント入力欄が表示される
  - submit ボタンを押すとコメントが保存される
  - ファイル名:行番号 コメント内容 の形式で全てのコメントが保存される

# 設計
-  yo を使って VS Code 拡張機能のスキャフォールディングを作成する
-  テストを書くこと。
-  README.md に使い方を記載すること。
- features ディレクトリを作って機能ごとにファイルを分けること。
